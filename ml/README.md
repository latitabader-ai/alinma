# نموذج تصنيف مخاطر التمويل — Random Forest

هذا المجلد يحتوي محرّك تقييم المخاطر الذي تحاكيه واجهة "التمويل الجماعي" في التطبيق.

## الملفات
- **`tamweel_data.csv`** — بيانات التدريب: 800 طلب تمويل اصطناعي بمعايير المخاطر السعودية (العمر، جهة العمل، الراتب، الالتزامات، التصنيف الائتماني، نسبة عبء الدين... ← مستوى الخطر).
- **`model_tuned.py`** — تدريب نموذج Random Forest مع ضبط المعاملات (GridSearchCV + تحقق متقاطع 5 طيّات)، وتقييمه، وتجربته على طلبات جديدة.

## التشغيل
```bash
pip install scikit-learn numpy
cd ml
python3 model_tuned.py
```

## الخصائص المستخدمة
- **رقمية:** العمر، مدة التوظيف، الراتب الشهري، الالتزامات الشهرية، التصنيف الائتماني (سمة)، مبلغ التمويل، مدة السداد، القسط الشهري، نسبة عبء الدين.
- **فئوية:** جهة العمل، نوع السلعة.
- **الهدف:** مستوى الخطر (منخفض / متوسط / مرتفع).

## العلاقة بالتطبيق
واجهة `src/pages/CrowdFinance.tsx` (تبويب "كيف يقيّم؟") تعرض العوامل وأوزانها المستوحاة من هذا النموذج. دالة `assess()` في الواجهة تحاكي منطقه لعرض فوري بدون خادم.
